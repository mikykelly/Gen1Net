1. How to use "BusMaster_v3.2.2_Gen1Net_Patch.zip"
1.1 After install the "BUSMASTER_Installer_Ver_3.2.2.exe", copy all the files to the folder("BUSMASTER.exe").
1.2 Select "Gen1Net CAN" in "Driver Selection"


2. How to use "Gen1Net_ProF_V1.0.zip"
1.1 You should install the INCA 7.1 or INCA 7.2.
1.2 If you want to support BUSMASTER(Gen1Net CAN), then select the Action: "EraseEs592Data".
1.3 If you want to support INCA, then select the Action: "ProgramES592Data".
